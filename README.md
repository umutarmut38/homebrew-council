# homebrew-council

Homebrew tap for [council](https://github.com/umutarmut38/council) — a
multi-agent orchestration TUI for competing AI coding-agent CLIs.

## Install

```bash
brew install umutarmut38/council/council
```

Or tap first, then install:

```bash
brew tap umutarmut38/council
brew install council
```

Upgrade with `brew upgrade council`.

## Notes

`Formula/council.rb` is generated automatically by council's release workflow.
Do not edit it by hand.
